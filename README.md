# Laptop_stand_3Dmodel
一个笔记本电脑的支架的3D模型，使用Solidworks制作。

## 💭有什么用：
```
1.正如你所看到的，这是一个电脑支架。
2.正如你想的那样，就是这样用的。
```

## 🤷怎么用：
```
1.将全部零件的文件克隆到本地。
2.将所有文件使用solidworks打开。
3.将文件导出为STL文件类型。
4.将文件导入到例如“Repetier-Host”的软件里面。
5.开始3D打印。
```
